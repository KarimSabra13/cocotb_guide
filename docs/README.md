# Docs Policy

This folder is mostly local-only and is intentionally ignored by Git.

Tracked from `docs/`:

- final generated PDF files
- markdown files such as README files

Ignored from `docs/`:

- LaTeX chapter sources
- front matter and config files
- tooling code under `tools/`
- figure specs and generated intermediates
- temporary build artifacts

Use this folder freely for authoring, but expect Git to keep only the final PDF output and markdown documentation.
